# AI开发平台高保真交互原型

## 项目概述

这是一个基于HTML5 + Tailwind CSS + 原生JavaScript的AI开发平台高保真交互原型系统，严格遵循黑白极简主题的UI设计规范。

## 技术栈

- **前端框架**: HTML5 + Tailwind CSS
- **交互逻辑**: 原生JavaScript (ES6+)
- **设计风格**: 黑白极简主题 (参考shadcn/ui)
- **响应式**: 支持桌面端和移动端

## 设计特色

### 🎨 黑白极简主题
- 主色调: 纯黑 (#000000) 和纯白 (#ffffff)
- 中性色: 现代灰度色阶
- 功能色: 仅保留必要的状态色 (成功绿、警告橙、错误红)
- 微妙阴影: 轻微的阴影效果营造层次感

### 📱 响应式设计
- 桌面端: 侧边栏 + 主内容区布局
- 移动端: 可折叠侧边栏 + 全屏内容
- 断点适配: 支持多种屏幕尺寸

### 🔄 交互体验
- 平滑过渡动画
- 悬浮效果
- 状态反馈
- 键盘导航支持

## 使用说明

### 本地运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器:
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   
   # 使用PHP
   php -S localhost:8000
   ```

### 功能演示

1. **导航切换**: 点击左侧导航菜单切换不同模块
2. **任务管理**: 在工作台模块中可以勾选/取消任务
3. **插件管理**: 查看AI插件状态和使用统计
4. **代码生成**: 在开发流程模块中体验AI代码生成
5. **响应式**: 调整浏览器窗口大小查看响应式效果

### 移动端体验

- 在移动设备或浏览器开发者工具中切换到移动端视图
- 点击左上角菜单按钮打开/关闭侧边栏
- 支持触摸手势操作
