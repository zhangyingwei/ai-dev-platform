使用说明

基本操作

  1. 1. 系统自动以"领导"角色进入
  2. 2. 浏览仪表板了解概况和快捷操作
  3. 3. 使用右上角角色选择器切换角色
  4. 4. 进入项目管理查看项目
  5. 5. 点击项目卡片进入详情页面
  6. 6. 切换标签页体验不同功能模块

特色功能

  1. 1. 仪表板 → 快捷操作 → 角色相关功能
  2. 2. 角色切换 → 实时权限变更
  3. 3. 需求管理 → 新建需求 → 录音功能
  4. 4. 输入需求描述 → AI辅助拆分
  5. 5. 项目详情 → 开发页面 → 需求+任务双模块
  6. 6. 子需求转换 → 开发任务 → 配置开发环境
  7. 7. 开发详情 → 代码编辑器 → 实时监控
  8. 8. 代码/部署/监控 → 项目集成管理
