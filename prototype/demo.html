AI开发平台演示指南

完整的功能演示和操作说明

🚀 快速开始

1 打开 index.html 文件
2 选择角色登录（推荐先选择"领导"角色体验完整功能）
3 按照下面的演示流程体验各个功能模块

提示： 用户名和密码可以任意输入，系统会自动登录。

👔 销售/售前角色演示

• 登录时选择"销售/售前"角色
• 进入需求管理页面
• 点击"新建需求"按钮
• 体验录音功能（需要麦克风权限）
• 查看语音转文字效果
• 使用AI辅助拆分需求功能
• 保存需求并查看需求列表

👑 领导角色演示

• 登录时选择"领导"角色
• 查看仪表板全局概览
• 进入需求管理审批需求
• 进入项目管理查看项目状态
• 点击项目卡片进入项目详情
• 切换不同标签页查看项目各方面信息

🏗️ 项目详情功能演示

概览标签页

  • • 项目进度可视化
  • • 最近活动时间线
  • • 项目统计数据
  • • 团队成员信息

代码标签页

  • • Git仓库管理
  • • AI编程工具集成
  • • 代码活动记录
  • • 工具连接状态

部署标签页

  • • 多环境状态监控
  • • 部署历史记录
  • • 一键部署操作
  • • 发布策略配置

监控标签页

  • • 系统关键指标
  • • 资源使用率
  • • 实时告警信息
  • • 性能监控图表

需求标签页

  • • 项目需求列表
  • • 需求状态跟踪
  • • 优先级管理
  • • 工时统计

团队标签页

  • • 团队成员详情
  • • 工作量统计
  • • 任务完成情况
  • • 成员管理操作

✨ 特色功能体验

🎤 录音转文字

1. 在需求收集页面点击"开始录音"

2. 允许浏览器访问麦克风权限

3. 说话录制需求内容

4. 点击"停止录音"查看转换结果

💡 系统会自动将语音转换为文字并填入需求描述

🤖 AI需求拆分

1. 在需求描述中输入详细需求

2. 点击"AI辅助拆分"按钮

3. 查看AI分析的拆分建议

4. 每个子需求包含优先级和工时估算

💡 AI会智能分析需求并提供专业的拆分建议

🔧 技术特点

📱

响应式设计

适配桌面端和移动端，提供一致的用户体验

💾

本地存储

使用IndexedDB实现数据持久化，支持离线操作

🎨

现代UI

基于TailwindCSS的黑白色系设计，简洁专业

💡 操作提示

  • • 所有数据都存储在浏览器本地，刷新页面数据不会丢失
  • • 录音功能需要HTTPS环境或localhost才能正常工作
  • • 不同角色登录会看到不同的功能菜单
  • • 项目详情页面集成了代码、部署、监控等所有功能
  • • 可以尝试切换不同角色体验不同的功能权限
